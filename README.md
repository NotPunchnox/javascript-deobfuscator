# javascript-deobfuscator
A simple program to unobfuscate javascript

coffeescript:
  `npm install coffee-script -g -d`
modules:
  `npm install`

put your code in './input.js' ( preferably minify )

start ( js ):
`node .` or `npm run js`

start ( coffeescript ):
`coffee main.coffee` or `npm run start`
